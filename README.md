# Kai Plugin Marketplace

Plugin catalog for Kai Desktop.

## About

This repository hosts `marketplace.json` — a catalog of plugins available for installation through Kai's built-in plugin marketplace.

## Plugins

| Plugin | Version | Description |
|--------|---------|-------------|
| [cron](https://github.com/LegionIO/kai-plugin-cron) | v0.1.13 | Schedule recurring tasks and reminders |

## Schema Version

The `schemaVersion` field at the root of `marketplace.json` indicates the catalog format version. Current version: `1`.
